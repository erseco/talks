La IA puede programar, pero la libertad hay que cuidarla

El giro: software libre

¿Qué libertad hay que cuidar?

Cuando ese desarrollo ocurre en el ámbito del software libre, aparecen preguntas específicas.

Las cuatro libertades (usar, estudiar, modificar, distribuir) dependen de que el código tenga una procedencia y una licencia limpias.

Si no sabemos de dónde viene el código que genera la IA, no sabemos si podemos ejercer esas libertades.
