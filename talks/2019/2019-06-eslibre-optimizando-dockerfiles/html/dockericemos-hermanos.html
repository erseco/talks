De Ballena a Gallopedro. Optimizando Dockerfiles

Dockericemos hermanos

Diferentes formas de dockerizar

Dockericemos hermanos

Veamos diferentes formas de dockerizar la misma aplicación y cómo, en cada iteración, vamos reduciendo el tamaño de la imagen y el tiempo de construcción.

1. Imagen base debian:stretch

1. Punto de partida: debian:stretch

FROM debian:stretch AS example
COPY . .
RUN apt-get update && \
    apt-get install -y \
        python3 \
        python3-pip
RUN pip3 install -r requirements.txt
RUN python3 test.py
  • Tamaño: 515 MB
  • Tiempo: 57 s

2. debian:stretch-slim y caché de capas

2. Versión slim y aprovechar la caché

Usamos debian:stretch-slim y copiamos primero requirements.txt para aprovechar la caché de capas antes de copiar el resto del código.

FROM debian:stretch-slim AS example
RUN apt-get update && \
    apt-get install -y \
        python3 \
        python3-pip
COPY requirements.txt .
RUN pip3 install -r requirements.txt
COPY . .
RUN python3 test.py
  • Tamaño: 427 MB
  • Tiempo: 51 s

3. Imagen base python:3.6-alpine

3. Saltamos a Alpine: python:3.6-alpine

FROM python:3.6-alpine AS example
RUN apk add --no-cache \
        build-base \
        zlib-dev \
        jpeg-dev \
        libpng-dev
COPY requirements.txt .
RUN pip3 install -r requirements.txt
COPY . .
RUN python3 test.py
  • Tamaño: 287 MB
  • Tiempo: 40 s

4. Paquetes virtuales y limpieza

4. Dependencias de compilación como paquete virtual

Instalamos las dependencias de compilación en un paquete virtual (.build-deps) y las borramos en la misma capa, dejando solo lo necesario para ejecutar.

FROM python:3.6-alpine AS example
COPY requirements.txt .
RUN apk add --no-cache jpeg libpng && \
    apk add --no-cache --virtual .build-deps \
        build-base \
        zlib-dev \
        jpeg-dev \
        libpng-dev && \
    pip3 install -r requirements.txt && \
    apk del .build-deps --force-broken-world
COPY . .
RUN python3 test.py
  • Tamaño: 123 MB
  • Tiempo: 34 s

5. alpine:3.9 con py3-pillow

5. Alpine puro con py3-pillow del repositorio

Partimos de alpine:3.9 e instalamos Pillow ya empaquetado (py3-pillow), evitando compilar nada.

FROM alpine:3.9 AS example
RUN apk add --no-cache \
        python3 \
        py3-pillow
COPY requirements.txt .
RUN pip3 install -r requirements.txt
COPY . .
RUN python3 test.py
  • Tamaño: 64,9 MB
  • Tiempo: 6 s
