Honeypot Huawei HG556a

One more thing... un honeypot WiFi

Durante las jornadas se montó un honeypot (un punto de acceso WiFi trampa) con un viejo router Huawei HG556a, anunciado con un nombre tentador: JASYP_FR33_WIFI. Un buen recordatorio de los riesgos de conectarse a redes abiertas que no controlamos.

Especificaciones:

  • CPU Broadcom BCM6358SKFBG a 300 MHz
  • 64 MB de RAM
  • 16 MB de Flash
  • OpenWRT 18.06.2
  • SSID: JASYP_FR33_WIFI

Configuración disponible en: github.com/erseco/openwrt-wifi-honeypot

Con la colaboración de @chelunike y @mati3.
